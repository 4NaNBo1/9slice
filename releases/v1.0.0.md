# v1.0.0

首次公开发布：Figma / MasterGo 双平台九宫缩放插件。

## 主要功能

- 读取选中图层的图片填充，在 UI 中显示切线预览。
- 支持 `top` / `right` / `bottom` / `left` 切片值输入，以及预览区拖拽调整。
- 按原图像素裁切 9 个区域，生成可缩放组件。
- 四角固定、边缘单向拉伸、中间双向拉伸。
- 底部显示作者与版本信息，并检查 GitHub 最新 release。

## 构建产物

- `dist/code.js`
- `dist/ui.html`
- `dist/manifest.json`
- `dist/manifest.mastergo.json`
